前言

最近也是在工作之余开发一个Android App，采用当前比较火的框架retrofit+rxjava+rxbus+lamda+greenDao+glide等。有兴趣的小伙伴可以关注一下我的github开源项目。本文主要介绍的是如何利用python抓取糗百的热门段子。

开发环境

  2. Python 2.7.10
  3. Node 7.8.0

编译器：

  1. python–>Pycharm Ec
  2. node–>Visual Code Studio
